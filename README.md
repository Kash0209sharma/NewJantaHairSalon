# New Janta Hair Salon

A modern, responsive website for New Janta Hair Salon built with React, Vite, and Tailwind CSS.

## 🌟 Features

- **Responsive Design**: Optimized for all devices (mobile, tablet, desktop)
- **Modern UI**: Clean and elegant interface with smooth animations
- **Service Showcase**: Display of available hair services
- **Gallery**: Photo gallery of salon work
- **Testimonials**: Customer reviews and feedback
- **Booking System**: Integrated appointment booking modal
- **Contact Information**: Easy access to salon details

## 🛠️ Tech Stack

- **Frontend**: React 18, Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: React Icons
- **Backend**: Node.js with Express (for appointments API)

## 🚀 Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Kash0209sharma/NewJantaHairSalon.git
   cd NewJantaHairSalon
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:5173](http://localhost:5173) in your browser.

### Backend Setup

The project includes a Node.js backend for handling appointments:

1. Navigate to the backend directory:
   ```bash
   cd backend
   npm install
   npm start
   ```
